Rock Paper Scissors

A simple game of rock-paper-scissors that displays both the user's and the computer's choices. The game continues until either the user or the computer reaches 5 points, while also keeping track of ties.

This project is built with vanilla HTML, CSS, and JavaScript. The logic took some time to work out, but essentially, a main game function manages the scores and enables the player buttons. Event listeners set the player's choice, while a function determines the computer's choice using Math.random. And the last notable function, checks the winner of each round and displays the result.
